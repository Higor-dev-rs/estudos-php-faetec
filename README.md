## 🚀 Atividades Práticas - FAETEC (Programador Web II)

Este repositório contém o conjunto de exercícios e desafios práticos desenvolvidos durante o curso de Programador Web II da FAETEC. O foco principal destas atividades é o domínio da linguagem PHP no lado do servidor (Back-end) e a manipulação dinâmica de dados.

---

## 📝 Sobre o Projeto

As atividades aqui presentes simulam situações reais de um sistema de gerenciamento de produtos, aplicando conceitos fundamentais de lógica de programação e engenharia de software que estou estudando na minha graduação em Análise e Desenvolvimento de Sistemas (ADS).

---

## 🛠️ Tecnologias e Conceitos Aplicados

- **PHP 8+**: Linguagem principal para o processamento lógico.

- **Parâmetros GET**: Captura de dados diretamente da URL para controle de estado da página.

- **Manipulação de Arrays**: Uso de arrays multidimensionais para simular bancos de dados.

- **Busca e Filtros**: Implementação de sistemas de busca parcial (usando mb_stripos) e filtros por categoria.

- **Ordenação Dinâmica**: Uso da função usort com o operador spaceship (<=>) para ordenar por preço e nome.

- **Paginação de Dados**: Cálculo de offset e uso de array_slice para exibição fragmentada de registros.

---

## 📂 Organização do Repositório

- **`projeto_consulta/`**: Pasta contendo um mini sistema integrado com busca e indexação.

- **`paginacao.php`**: Script focado na lógica de divisão de resultados por páginas.

- **`ordem.php` & `filtro.php`**: Arquivos dedicados à organização e seleção de dados de arrays.

- **`acoes.php`**: Lógica de processamento de requisições.
